Court Auction Console

Inspect slots, place bids, and approve winners from one screen.

This page talks directly to the FastAPI backend so you can verify the auction logic without building Telegram, WhatsApp, or a separate web app first.

API
GET /slots
POST /slots/{id}/bids
POST /admin/slots/{id}/approve

User Controls

To place a bid, enter a user ID and use increments like 10.50, 11.00, 11.50.

Admin Controls

When the token is valid, cards in CLOSED_PENDING_ADMIN state will show approve and reject actions.
